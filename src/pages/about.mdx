This project combines open-source technologies and commercial services, with all custom code released under the MIT license. 
The full source code is public and maintained on GitHub in two repositories:

- [n8n-with-ollama-for-kubernetes](https://github.com/acosoft/n8n-with-ollama-for-kubernetes)
- [timeoff-website](https://github.com/acosoft/timeoff-website)

Below is a list of the technologies and resources used:

## Core Technologies

- [Docusaurus](https://docusaurus.io/) – Static site generator for modern documentation
- [cal.com](https://cal.com/) – Calendar and booking integration
- [n8n](https://n8n.io/) – Workflow automation platform
- [Ollama](https://ollama.com/) – Open-source LLM runtime
- [phi3:mini](https://ollama.com/library/phi3:mini) – Lightweight open-source language model from Microsoft
- [k3s](https://k3s.io/) – Lightweight Kubernetes distribution for servers
- [GitHub](https://github.com/) – Code hosting and collaboration
- [GitHub Actions](https://github.com/features/actions) – Automated deployment workflows
- [Hetzner Cloud](https://www.hetzner.com/cloud) – Cloud infrastructure provider


## Cloudflare

- [Cloudflare](https://www.cloudflare.com/) – Security and performance platform
- [Cloudflare Workers](https://workers.cloudflare.com/) – Docusaurus site hosting
- [Cloudflare Zero Trust](https://www.cloudflare.com/products/zero-trust/) – Secure access and identity management
- [Cloudflare Access](https://www.cloudflare.com/products/zero-trust/access/) – Zero Trust access control
- [Cloudflared](https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/install-and-setup/installation/) – Secure tunneling and private network access
- [Cloudflare R2](https://www.cloudflare.com/products/r2/) – S3-compatible object storage for workflow bootstrap and assets


## Mailtrap

- [Mailtrap Transactional Email API](https://mailtrap.io/pricing/?tab=email-api) – Transactional email sending
- [Mailtrap Email Sandbox](https://mailtrap.io/pricing/?tab=email-sandbox) – Email testing and sandboxing


## Development

- [pnpm](https://pnpm.io/) – Fast, disk space efficient package manager
- [Node.js](https://nodejs.org/) – JavaScript runtime
- [VS Code](https://code.visualstudio.com/) – Code editor
- [Docker Compose](https://docs.docker.com/compose/) – Multi-container application orchestration
- [Docker](https://www.docker.com/) – Containerization platform
- [Kubernetes](https://kubernetes.io/) – Container orchestration platform
- [Helm](https://helm.sh/) – Kubernetes package manager for deployment
- [Terraform](https://www.terraform.io/) – Infrastructure as code for automating GitHub Actions and variable management
- [React](https://react.dev/) – JavaScript library for building user interfaces
- [TypeScript](https://www.typescriptlang.org/) – Typed superset of JavaScript
- [GitHub Copilot](https://github.com/features/copilot) – AI-powered coding assistant
- [ChatGPT 4.1 (OpenAI)](https://openai.com/chatgpt) – AI language model via GitHub Copilot
